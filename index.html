문법 트리 생성기

들여쓰기 텍스트로 SVG 문법 트리를 즉시 만들고,
인공어 설계 도구로 나만의 언어를 구상하세요.

사이트 열기
SVG 트리 생성들여쓰기 입력으로 실시간 문법 트리 시각화
음소 목록자음·모음 IPA 기호로 음운 체계 구성
음운 변화 규칙환경별 음운 규칙 정의 및 어휘 자동 적용
문자 체계음소에 고유 문자를 대응시켜 표기 체계 설계
유형론 설정어순·형태론·핵어 방향 등 언어 유형 지정
어휘집 관리단어·의미·품사 등록 및 파일 내보내기
개발 서버가 실행 중이지 않다면 프로젝트 루트에서
python start.py 를 실행하세요.
